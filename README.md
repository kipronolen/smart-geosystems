# Smart GeoSystems

A premium static website for Smart GeoSystems featuring geospatial services and the SentMailTracker SaaS product.

## Features

- **Professional Design**: Modern blue and green gradient theme
- **Responsive Layout**: Works perfectly on all device sizes
- **Smooth Animations**: CSS transitions and JavaScript interactions
- **Geospatial Services**: GIS, Engineering Surveying, CAD Drafting
- **SaaS Product**: Dedicated section for SentMailTracker
- **Premium UI/UX**: Professional design with modern aesthetics

## Services Offered

1. **GIS & Geospatial Services**
   - Spatial Data Analysis
   - Interactive Web Mapping
   - Remote Sensing
   - Location Intelligence

2. **Engineering Surveying**
   - Topographic Surveys
   - Construction Staking
   - Boundary Surveys
   - GPS & Drone Surveying

3. **2D & 3D CAD Drafting**
   - Architectural Drafting
   - 3D Modeling & Rendering
   - Technical Documentation
   - BIM Integration

## SaaS Product: SentMailTracker

An innovative Gmail extension for email tracking with features like:
- Real-time email tracking
- Read receipts and open notifications
- Link click monitoring
- Dashboard analytics

## Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/smart-geosystems.git
cd smart-geosystems
```

2. Install dependencies:
```bash
npm install
```

3. Start the server:
```bash
npm start
```

4. Open your browser and visit:
```
http://localhost:3000
```
